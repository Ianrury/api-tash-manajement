
[## Dokumentasi Backend Task Management]

### Deskripsi Singkat Aplikasi
Aplikasi ini merupakan backend untuk sistem manajemen tugas yang memungkinkan pengguna untuk membuat, mengelola, dan memantau tugas. Terdapat fitur autentikasi, manajemen user, serta dokumentasi API menggunakan Swagger.

### Langkah Menjalankan Backend
1. Pastikan Node.js dan npm sudah terinstal di komputer Anda.
2. Clone repository ini dan masuk ke folder backend.
3. Install dependencies dengan perintah:
	```bash
	npm install
	```
4. Jalankan server dengan perintah:
	```bash
	npm start
	```
5. Backend akan berjalan di port yang telah ditentukan (default: 3000).

### Teknologi yang Digunakan
- Node.js
- Express.js
- Sequelize (ORM)
- PostgreSQL (Database)
- JWT (JSON Web Token) untuk autentikasi
- Swagger untuk dokumentasi API
